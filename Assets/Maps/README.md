# The format for map data

## Map array
0 ~ 6 : Normal Tiles
  0 : None
  1 : Black
  2 : White
  3 : Red
  4 : Blue
  5 : Yellow
  6 : Green
10 ~ 16 : Wall Tiles
  10 : Wall with no color
  11 : Black Wall
  12 : White Wall
  13 : Red Wall
  14 : Blue Wall
  15 : Yellow Wall
  16 : Green Wall

## Monster data
Specify the type of monster in field "name"

## Orb data
Specify the type of color in field "color"
색깔 이름:
"none", "black", "white", "red", "blue", "yellow", "green"
